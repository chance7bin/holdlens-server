# HoldLens Server Domain Language

HoldLens server is the long-lived business fact source for user assets, public fund data, market quote targets, and processing task state.

## Language

**当前持仓**:
用户在某个资产账户下真实持有的资产事实，包含账户、资产、金额、币种、来源和状态等信息。
_Avoid_: 仅凭代码导入形成的持仓、关注项

**关注资产**:
用户主动导入用于跟踪和刷新公开数据的基金或股票资产主数据；第一版不代表用户已经持有该资产。
_Avoid_: 当前持仓、交易记录

**基金目录**:
HoldLens 全局已知的公开基金身份集合，包含基金代码、简称、类型和拼音检索信息，用于基金发现和详情查询；进入基金目录不代表用户持有、关注，也不代表系统需要定期执行单基金刷新。
_Avoid_: 当前持仓、关注资产、刷新目标

**基金目录同步**:
以外部目录本次成功返回的基金为增量覆盖范围；未返回的已有基金继续保留，不能仅因单次缺失而删除或停用。
_Avoid_: 全表替换、缺失即删除

**刷新目标**:
进入公开基金详情或股票行情主动刷新范围的代码目标，通常由用户持有、关注、近期访问或系统任务注册产生；完整基金目录默认不等于刷新目标。
_Avoid_: 当前持仓、基金目录

**近期查看基金**:
用户最后一次查看详情后的九十天内仍具有近期关注度的基金；它临时进入定时刷新范围，再次查看会延长窗口，过期后退出但不删除已有公开数据。
_Avoid_: 关注资产、永久刷新目标

**基金详情异步回源**:
基金详情查询优先返回 server 已保存的公开数据；数据缺失或过期时创建后台刷新，查询请求不等待外部基金接口完成。
_Avoid_: 查询时同步抓取、绕过 server 直连数据源

**基金收益快照**:
单只基金在最新数据日期下的当前净值与阶段收益事实；当前阶段只维护最新值，不表达每日历史序列。
_Avoid_: 基金收益历史、净值走势

**基金收益覆盖状态**:
基金类型是否属于当前收益数据源的支持范围；数据源未覆盖表示当前阶段没有该类收益能力，不属于刷新失败。
_Avoid_: 刷新失败、字段缺失

**基金当前前十大重仓**:
单只基金在最新公开披露季度中按排名选取的前十只股票持仓；当前阶段不表达更早季度、完整股票持仓或债券持仓。
_Avoid_: 基金完整持仓、基金持仓历史

**数据日期**:
公开数据源声明的数据所属日期或报告期，例如收益日期或重仓披露季度；它表达数据内容截至何时。
_Avoid_: 抓取时间、数据库更新时间

**抓取时间**:
系统实际完成某个基金数据切片获取的时间；目录、申购、收益和重仓分别记录，不能替代数据日期。
_Avoid_: 数据日期、统一更新时间

**业务市场**:
HoldLens 对资产所属公开市场的稳定业务分类，例如 A 股、港股或美股；它不绑定某个第三方行情源的接口编码。
_Avoid_: 东方财富市场编码、交易所代码

**交易所代码**:
资产在业务市场内的交易所或板块归属代码，例如 A 股内的上交所、深交所或北交所。
_Avoid_: 业务市场、第三方接口市场编号

**数据源市场编码**:
第三方行情源为了请求或解析数据使用的市场编码，例如东方财富 secid 中的市场编号；它只表达数据源适配语义，不作为 HoldLens 的业务市场分类。
_Avoid_: 业务市场、交易所代码

**美股**:
HoldLens 的业务市场之一，用于表达美国股票市场公开资产集合；在跨项目刷新契约和 `stock_market` 中建议使用稳定枚举 `US_STOCK`。东方财富 `105`、`106`、`107` 等编号只属于数据源市场编码。
_Avoid_: 东方财富市场编码、secid

**股票全量行情刷新**:
针对某个业务市场一次性刷新当前公开股票行情的处理任务。agent 负责抓取并回传结构化结果，server 负责任务状态、幂等、诊断和长期行情事实写入。
_Avoid_: 个股补刷、历史行情刷新、用户持仓刷新
